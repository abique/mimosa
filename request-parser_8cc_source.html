01022        function.

01024        the only way this function was invoked is if the default action

01033        consistent state with a default action.  There might have been a
01034        previous inconsistent state, consistent state with a non-default

01037        correct lookahead information, and it depends on the parser not

01042        one exception: it will still contain any token that will not be

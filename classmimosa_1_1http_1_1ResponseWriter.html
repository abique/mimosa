bool mimosa::http::ResponseWriter::sendHeader ( runtime::Time  timeout)

Writes the header to the client. If content_length_ is 0 then the content length is unspecified and the connection will close after this response (keep_alive_ will be set to false). Until you call sendResponseHeader, everything you write to body is buffered, so when you finishes Response knows the amount of data to be written and set content_length_
